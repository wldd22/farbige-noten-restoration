# Farbige Noten (1888-1889) Restoration

This repository is for a **documented digital restoration** of *Farbige Noten* (1888–1889) by *Ch. A. B. Huth*, published by *Verlagsanstalt und Druckerei A.G. (vormals J. F. Richter), Hamburg*.

The project aims to **preserve, analyse, and reconstruct** the work's textual, typographic, and visual structure, with particular attention to historical accuracy.


## Project Scope

The restoration intends to include:

- A **transcription** of all the original text
- **Translations** of all the original text
- **Reconstruction** and **restoration** of the original typography
- **Reconstruction** and **restoration** of the work's ornaments and layout
- **Documentation** of sources, methods, and decisions


## Repository Structure

```
/masters         - Original source scans
/transcriptions  - OCR and manual transcripts, and translations
/fonts           - Reconstructed font glyphs and fonts
/vectors         - Reconstructed ornaments and graphics
/layouts         - Reconstructed page layouts
/docs            - Documentation of research, decisions, and work
```
